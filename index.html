qAlgebra.jl

qAlgebra.jl provides symbolic tools for constructing and manipulating quantum operator expressions on structured composite systems of qubits, spins, and bosonic modes.

The core abstraction is the StateSpace, which defines:

  • the symbolic variables (e.g. coupling constants),
  • the operator bases (e.g. Pauli, ladder, raising/lowering),
  • and the indexed structure of composite systems.

Example: Constructing a State Space

using qAlgebra
qspace = StateSpace("alpha", "beta(t)", "gamma_i", "delta_i",
                    operators=["A(!i)", "B(U,H,i)"],
                    h=QubitPM(), i=(3, QubitPauli()), b=Ladder())
StateSpace: [β(t), γᵢ, γⱼ, γₖ, δᵢ, δⱼ, δₖ, α]
   - SubSpace ["h"]: PM Qubit (Fermionic):  pₚ, mₚ, zₚ, Iₚ (identity)
   - SubSpace ["i", "j", "k"]: Pauli Qubit (Fermionic):  xₚ, yₚ, zₚ, Iₚ (identity)
   - SubSpace ["b"]: Ladder (Bosonic):  p†, p
   - Op: A
   - Op: B(H,U)

You can access variables and operators from this space in several ways. By creating dictionaries of all variables, operators and abstract operators

var_dict, op_dict, abstract_dict = base_operators(qspace)

individual variables 

alpha = base_operators(qspace, "alpha")
beta  = base_operators(qspace, "beta")

individual subsystem bases (do_dict=true would return the entries as a dictionary):

ph, mh, zh = base_operators(qspace, "h", do_dict=false)
xi, yi, zi, pi, mi = base_operators(qspace, "i", do_dict=false)
xj, yj, zj, pj, mj = base_operators(qspace, "j", do_dict=false)
xk, yk, zk, pk, mk = base_operators(qspace, "k", do_dict=false)
b, n = base_operators(qspace, "b", do_dict=false)

or specific abstract operators (with or without indexes or as a function accepting an index):

A = base_operators(qspace, "A")
A1 = base_operators(qspace, "A_1")
As = base_operators(qspace, "A", do_fun=true)
A, A1, As(2)
(A, A₁, A₂)

Building, Printing and Modifying Expressions

You can build symbolic expressions using variables and operators:

expr = 2 * (alpha + beta) * 1im * xi + alpha * Dag(b) * xi * yi

\[2i \alpha\left( \hat{x}_{i}+2 \hat{z}_{i}\hat{b}^\dagger\right)+2i\beta(t) \hat{x}_{i}\]

You can get the string and latex_string expressions via 

string(expr)
latex_string(expr)

We provide a number of useful operations, such as sums

qsum = Sum("j", alpha * yi * yj + Sum("k", beta * alpha^2 * xi * xj * xk))

\[\sum_{j}^{=}\left(\alpha \hat{y}_{i}\hat{y}_{j}+\sum_{k}^{=}\beta(t)\alpha² \hat{x}_{i}\hat{x}_{j}\hat{x}_{k}\right)\]

Nested sums can be flattened

flatten(qsum)

\[\sum_{j}^{=}\alpha \hat{y}_{i}\hat{y}_{j}+\sum_{j,k}^{=}\beta(t)\alpha² \hat{x}_{i}\hat{x}_{j}\hat{x}_{k}\]

and you can enforce index distinction (i.e. $j \ne k$):

neq(qsum)

\[\alpha+2\beta(t)\alpha² \hat{x}_{i}+\sum_{j}^{\neq}\left(\alpha \hat{y}_{i}\hat{y}_{j}+\beta(t)\alpha^{2}\left( \hat{x}_{i}+ \hat{x}_{j}\right)\right)+\sum_{k}^{\neq}\beta(t)\alpha² \hat{x}_{k}+\sum_{(j,k)}^{\neq}\beta(t)\alpha² \hat{x}_{i}\hat{x}_{j}\hat{x}_{k}\]

You can define differential equations of operator expectation values

diff_eq = d_dt(zi, alpha * expr + qsum)

\[\frac{\text{d} \hat{z}_{i}}{\text{d}t} = \alpha+2i \alpha^{2}\left( \hat{x}_{i}+2 \hat{z}_{i}\hat{b}^\dagger\right)+2i\beta(t)\alpha(1-i\alpha) \hat{x}_{i}+\sum_{j}^{\neq}\left(\alpha \hat{y}_{i}\hat{y}_{j}+\beta(t)\alpha^{2}\left( \hat{x}_{i}+ \hat{x}_{j}\right)\right)+\sum_{k}^{\neq}\beta(t)\alpha² \hat{x}_{k}+\sum_{(j,k)}^{\neq}\beta(t)\alpha² \hat{x}_{i}\hat{x}_{j}\hat{x}_{k}\]

Operator Functions

We provide the following operator functions, with as of now limited support

qCommutator(Sum("i", alpha * ph * xi * yi) + zj, zh)

\[\left[ \hat{z}_{j}+\sum_{i}^{=}i\alpha \hat{p}_{h}\hat{z}_{i},\, \hat{z}_{h}\right]\]

exp(Sum("i", alpha * ph * xi * yi) + zj) + zh

\[ \hat{z}_{h}+\exp\left( \hat{z}_{j}+\sum_{i}^{=}i\alpha \hat{p}_{h}\hat{z}_{i}\right)\]

log(Sum("i", alpha * ph * xi * yi) + zj)

\[\log\left( \hat{z}_{j}+\sum_{i}^{=}i\alpha \hat{p}_{h}\hat{z}_{i}\right)\]

power(Sum("i", alpha * ph * xi * yi) + zj, 2) + zh

\[ \hat{z}_{h}+\left( \hat{z}_{j}+\sum_{i}^{=}i\alpha \hat{p}_{h}\hat{z}_{i}\right)^{2}\]

root(Sum("i", alpha * ph * xi * yi) + zj, 2) + zh

\[ \hat{z}_{h}+\sqrt{ \hat{z}_{j}+\sum_{i}^{=}i\alpha \hat{p}_{h}\hat{z}_{i}}\]
